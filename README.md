### Objective
to convert the existing Student Management CRUD ([LVCCWAD/student-ms](https://github.com/LVCCWAD/student-ms)) into a different management system (for example: Product Management, Book Management, Employee Management, etc.). The system should still perform basic Create, Read, Update, and Delete (CRUD) operations.

## Installation
- must have php and composer

```bash

```
```bash
php artisan serve
```
